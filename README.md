# Superstore Sales & Profit Dashboard

This project analyzes the Superstore dataset to uncover key business insights.

## Tools Used
- Python (Pandas)
- Excel
- Pivot Tables
- Data Visualization

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Total Customers

## Insights
- Identified loss-making subcategories
- Analyzed sales trends over time
- Explored regional performance
- Highlighted top customers
